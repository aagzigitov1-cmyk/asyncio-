День 4 — Rate Limiting и правила вежливости

🎯 Цель дня
Реализовать систему ограничения скорости запросов и соблюдения правил вежливости при краулинге.

📋 Требования

1. Класс RateLimiter
Создать класс для ограничения скорости:
- __init__(self, requests_per_second: float = 1.0, per_domain: bool = True)
- async def acquire(self, domain: str = None) — ожидание разрешения на запрос
- 🔄 Отдельные лимиты для каждого домена
- 🌍 Глобальный лимит (если per_domain=False)

2. Интеграция с AsyncCrawler
- ➕ Добавить RateLimiter в AsyncCrawler
- ⏱️ Применять лимиты перед каждым запросом
- ⚙️ Настраиваемые задержки между запросами

3. Парсинг robots.txt
Создать класс RobotsParser:
- async def fetch_robots(self, base_url: str) -> dict — загрузка robots.txt
- can_fetch(self, url: str, user_agent: str = "*") -> bool — проверка разрешения
- get_crawl_delay(self, user_agent: str = "*") -> float — получение задержки из robots.txt
- 💾 Кэширование правил для каждого домена

4. Соблюдение robots.txt
- 🔍 Проверять robots.txt перед каждым запросом
- 🚫 Уважать правила Disallow
- ⏰ Использовать задержки из Crawl-delay
- 📝 Логировать блокированные URL

5. Настраиваемые задержки
Реализовать:
- ⏱️ Минимальная задержка между запросами (min_delay)
- 🎲 Случайная задержка для имитации человеческого поведения (jitter)
- 📈 Экспоненциальный backoff при ошибках

6. User-Agent управление
- 🤖 Настраиваемый User-Agent
- 🔄 Ротация User-Agent (опционально)
- 📋 Соблюдение правил robots.txt для конкретного User-Agent

7. Мониторинг скорости
Отслеживать и выводить:
- ⚡️ Текущую скорость запросов (req/sec)
- ⏱️ Среднюю задержку между запросами
- 🚫 Количество заблокированных robots.txt URL

8. Демонстрация
Создать скрипт, который:
- 🌐 Краулит сайт с соблюдением robots.txt
- 🚦 Применяет rate limiting
- 📊 Показывает статистику по задержкам
- 🚫 Демонстрирует блокировку запрещённых URL

Пример конфигурации:
```python
crawler = AsyncCrawler(
    max_concurrent=5,
    requests_per_second=2.0,  # 2 запроса в секунду
    respect_robots=True,
    min_delay=0.5,  # минимум 0.5 сек между запросами
    user_agent="MyBot/1.0"
)
```

9. Тестирование
- ✅ Протестировать rate limiting для одного домена
- ✅ Протестировать rate limiting для разных доменов
- ✅ Протестировать парсинг robots.txt
- ✅ Протестировать блокировку запрещённых URL
- ✅ Проверить соблюдение задержек

🎉 Критерии успеха
- ✅ Rate limiting работает корректно
- 📋 robots.txt парсится и соблюдается
- ⏱️ Задержки между запросами соблюдаются
- 🚫 Запрещённые URL блокируются
- 📊 Статистика по задержкам отображается
