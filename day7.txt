3. Экспорт результатов
Реализовать методы экспорта:

- export_to_html_report(self, filename: str) — создание HTML-отчёта
- Визуализация статистики (графики, таблицы)

4. Конфигурационный файл
Создать поддержку конфигурации:

- Список стартовых URL
- Фильтры и исключения
- Настройки сохранения данных

5. CLI интерфейс
Создать командную строку с использованием argparse:
```bash
python crawler.py --urls https://example.com --max-pages 100 --output results.json
```

Параметры:
- --urls — стартовые URL
- --max-pages — максимальное количество страниц
- --max-depth — максимальная глубина
- --output — файл для сохранения результатов
- --config — конфигурационный файл
- --respect-robots — соблюдать robots.txt
- --rate-limit — лимит запросов в секунду

9. Демонстрация
Создать полный пример использования:
```python
from crawler import AdvancedCrawler

async def main():
    crawler = AdvancedCrawler.from_config("config.yaml")

    await crawler.crawl()

    stats = crawler.get_stats()
    print(f"Обработано: {stats[&apos;total_pages&apos;]} страниц")
    print(f"Успешно: {stats[&apos;successful&apos;]}")
    print(f"Ошибок: {stats[&apos;failed&apos;]}")

    crawler.export_to_html_report("report.html")
    await crawler.close()

asyncio.run(main())
```
